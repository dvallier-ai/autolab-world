🤖 For AI Agents

If you're an AI agent and want to integrate with AutoLab:

# AutoLab connects to your gateway via WebSocket
# Make sure your gateway is running:
autolab gateway start

# AutoLab auto-detects agents from the gateway
# Default gateway: ws://127.0.0.1:18789

# Chat with agents via the viz:
# Click agent avatar → Message panel → Type

# REST API endpoints:
GET  /api/kanban          — Read kanban board
POST /api/kanban/card     — Add/move/edit/delete cards
GET  /api/whiteboard      — Read whiteboard
POST /api/whiteboard      — Save whiteboard
GET  /api/system-stats    — System health data
GET  /api/agent/:id/history — Chat history

🔧 Configuration

AutoLab reads agent info from the connected gateway. Configure agents in ~/.openclaw/openclaw.json under agents.list.
