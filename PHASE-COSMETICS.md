# v0.4.5 — Visual Variety & Cosmetics

**Goal:** Make agents visually distinct beyond just color - different shapes, materials, effects per agent.

## Agent Visual Diversity

### Geometry Options
- **Nova:** Dodecahedron (12 faces) — complex, mystical
- **Liam:** Cube — solid, reliable, infrastructure-focused
- **Paradox:** Octahedron (8 faces) — dynamic, transitional
- **Future agents:** Tetrahedron, Icosahedron, custom meshes

### Material Themes
- **Metallic agents:** Chrome, gold, copper finishes
- **Glass/crystal:** Transparent with internal glow
- **Holographic:** Shifting iridescent colors
- **Matte:** Flat colors for minimalist look

### Per-Agent Effects
- **Nova:** Floating particles around (mystical aura)
- **Liam:** Rotating data streams (tech vibe)
- **Paradox:** Glitch/flicker effect (reality bending)

### Environmental Polish
- **Ground variations:**
  - Circuit board pattern option
  - Tron-style grid lines
  - Nebula/space backdrop
  - Procedural terrain
- **Lighting:**
  - Dynamic day/night cycle
  - Spotlight on active agent
  - Ambient color shifts based on system state

### Agent Accessories
- **Status indicators:**
  - Crown/halo when speaking
  - Chain links when connected to other agents
  - Warning signs when errors occur
  - Tool icons floating nearby during tool calls

## Configuration
- Agent appearance stored in config file
- Hot-reload on config change
- UI panel to switch themes live

## Timeline
- 1-2 hours for geometry variety
- 1 hour for material experiments
- 1 hour for per-agent effects
- 30min for ground/lighting polish

**Total estimate: 4-5 hours**
